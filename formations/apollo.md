# Formation Apollo

Le but de cette formation est de voir les standards en terme de workflow d'utilisation de Apollo dans une application React.

Vous allez retrouver plusieurs cas d'usage qui va falloir résoudre, en vous assurant que les tests Jest passent sur les composants concernés.

## Introduction

Vous êtes une agence web et vous êtes missionée pour développer une application qui va gérer l'ensemble du catalogue d'un V&B (bar à bières, vins et tapas).


## Exercices
